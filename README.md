# we🌦️ Weather Prediction System using Machine Learning
📌 Overview
The Weather Prediction System is a Python-based machine learning project that predicts whether it will rain or not based on weather conditions such as temperature, humidity, wind speed, and atmospheric pressure.
This project demonstrates how data analysis and classification algorithms can be applied to real-world weather forecasting problems.
🚀 Features
📥 Load weather dataset (CSV format)
🧹 Data cleaning and preprocessing
📈 Analyze weather patterns
📊 Data visualization (graphs & charts)
🤖 Rain prediction using Machine Learning
📉 Model evaluation (accuracy & confusion matrix)
🌦️ Real-time sample prediction
🧠 Tech Stack
Programming Language: Python 🐍
Libraries:
Pandas
Matplotlib
Seaborn
Scikit-learn

Column Name
